# Amazon Price Tracker

This is a **Python-based Amazon price tracker** that monitors the price of a specific product — the *Christmas Wooden Nutcracker Soldier* — and sends an **email alert** when the price drops below a specified threshold. It also logs price history to a CSV file for historical tracking.

---
## Features
- Automatically checks the product price on Amazon.
- Runs every day to monitor price changes.  
- Sends an email notification when the price drops below a defined threshold.
- Logs daily price history to a CSV file for analysis.
